Ausführliche Beschreibung

Controller für Item-bezogene API-Endpunkte im LaendleGuessr Backend.

Dieses Modul enthält Funktionen zur Verwaltung und Abfrage von Items und deren Zuordnung zu Benutzern. Alle Funktionen sind für die REST-API vorgesehen und werden von OpenAPI genutzt.
